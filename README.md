# Aether SaaS Showcase

> **Stack:** Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · Framer Motion

A polished marketing site built as a portfolio-grade Next.js template. It demonstrates modern layout, motion design, and component architecture with a dark glassmorphic theme.

Key patterns:
- App Router with client-side motion and effects
- Responsive Tailwind-only styling
- Accessible color choices and semantic sections
- Vercel deploy ready

This repo is intentionally public as a showcase, not a production product.
